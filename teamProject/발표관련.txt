	- 인터페이스는 전체를 구체적으로 설명


서비스개요 : 질문도 하고 대화를 나누듯이 하기
		PT를 받아보신 분들은 아시겠지만 PT를 한번도 경험해보지 못한 분들은
		어떻게 시작해야하는지 잘 모르시죠?
		그런 분들을 위해 만들어진 사이트가 득근득근입니다.
		저는 PT를 받기 위해 헬스장에 방문했을 때 배정받은 트레이너가 마음에 들지 않아도 
		소심한 편이라 다른 분으로 교체를 못하고 그냥 받는 경우가 많은데 
		저처럼 소심한 분들이나 집에서 편하게 트레이너에 대한 정보를 미리 알고싶은 분들을 위해
		득근득근은 O2O 서비스 즉 온라인 투 오프라인 서비스를 제공합니다.

		기획일정표를 보시면 5주간 기간별로 어떠한 작업을 진행하였는지 보실 수 있구요

로직프로세스 : 한가지를 예를 들어서 손으로 가리키며 진행순서를 설명
	
ERD : 회원과 트레이너의 정보를 보관할 수 있도록 구성된 테이블들입니다.

User flow : 마찬가지로 하나씩 가리키며 순서를 대화하듯이 설명
		마무리는 다른 기능들도 이와 같이 되어있지만 특정 순서에 따라 진행됩니다.

인터페이스 : 간략하게 설명 하나하나 설명하지말고
		대략적인 기능들만 설명
		자세한 설명은 시연 때 다시 설명 드리겠습니다.
		로그인 : 회원들이 직접 로그인을 선택하는건 번거로울 것 같아서 
			자동으로 분기처리가 되도록 구성해놓음
		메인 > 회원가입 > 로그인 > 회원 마이 > 트레이너 마이 > 관리자 > 운동성향 > 검색 >
		> 전체 > 상세보기 > 결제 > 충전 > 리뷰 > 회원사진 > Q&A > 신고


-------------------------- 회원 계정으로 시작 --------------------------
1. 검색 후 검색 목록 보여주기 > logo 클릭 후 메인으로 이동
2. 운동성향 진행 > 여, 부천시 상동, 다이어트, 상담 후 결정 > logo 클릭 후 메인으로 이동
3. 트레이너 전체 목록 보여주기 > 임솔 트레이너 보러가기 클릭
4. 트레이너 상세 페이지 설명 
5. 구경하러 가기 클릭 > 버튼 없는거 말해주고 뒤쪽에서 자세한 설명
5. 상담하러 가기 클릭 > 카카오톡 보여주고 > 새창 닫기
6. 결제하러 가기 클릭 > 충전 > 충전 완료 > 결제 페이지 > 결제 > 결제 완료
    > 회원 마이페이지 > 리뷰 작성 > 첨부파일까지 등록(bewforeafter.jpg, fitnessCenter.jpg) > 리뷰 게시판 
    > 트레이너 상세보기 > 수정 > 삭제 > 리뷰보기 > 리뷰 상세보기
-------------------------- 트레이너 계정으로 전환 --------------------------
7. 트레이너 마이페이지 > sns 이동 > 자세한 설명 > 프로필 사진 변경(profile.jpg) > 사진 업로드(sns.jpg)
    > sns.jpg 삭제 > 우측 상단 마이페이지 클릭해서 다시 이동
8. 트레이너 마이페이지 > 프로필 변경 확인 > 아이디, 이름, 비밀번호는 생략 > 결제 내역(환전x 결제만 상세보기 클릭)
    > 환전하기 직접 입력 > 100,000 > 신한은행, 000-000-000000, aaa111!!! > 결과창 확인
    > PT 가격, 경력 수정, 컨텐츠 작성, URL 수정 변경하지 않고 확인만 시켜주고 취소
9. 관리자 페이지 > 답변 완료 없는 글 선택 > 답변 > 구분선 밑에 아무거나 작성 > 답변 > 상세보기에서 확인
    > 우측 상단 Report > 처리 완료 없는 글 선택 > 관리자 > ironman 입력 > 차단 > 조치하기 > 상세보기에서 확인
